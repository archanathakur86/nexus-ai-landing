# Nexus AI - Next Gen Landing Page

A futuristic, high-performance SaaS landing page built with React.js, Tailwind CSS, and Framer Motion.

## Live Demo
[View Live Project Here](https://nexus-ai-landing-urc2.vercel.app/)

---

## Features

* **Modern Aesthetic:** Deep dark mode theme with glassmorphism effects and glowing gradients.
* **Bento Grid Layout:** Innovative feature showcase using a masonry-style grid layout.
* **Smooth Animations:** Entrance animations and hover effects powered by Framer Motion.
* **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop screens.
* **Component Based:** Clean and modular React code structure.

---

## Tech Stack

* **Framework:** React.js (Vite)
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Icons:** Lucide React
